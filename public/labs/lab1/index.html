CSS — язык описания внешнего вида документа. Стили подключают через <link>, <style> или атрибут style.

Каскад позволяет переопределять стили, а специфичность определяет приоритет.

  1. Текст
    1. Шрифт
      1. font-size
      2. font-weight
    2. Цвет
      1. color
      2. opacity
  2. Блок
    1. Размеры
      1. width
      2. height

Каскадность: побеждает правило с наибольшей специфичностью. inline > ID > класс > тег.

Блочная модель (box model): content + padding + border + margin. box-sizing: border-box включает padding и border в ширину элемента.

!important перекрывает все правила каскада. Злоупотреблять не стоит — усложняет поддержку.
